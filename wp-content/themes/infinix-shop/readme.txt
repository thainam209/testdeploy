== Description ==

Infinix Shop is a complete WordPress Full Site Editing theme that gets makes your site building super easy. This theme is compatible with WooCommerce, which allows you to effortlessly set up an online store, showcasing your products in an elegant and clutter-free layout. the theme's efficient performance ensures that your website loads quickly, contributing to a positive user experience and search engine optimization benefits. With Infinix Shop, you can rest assured that your site will not only look great but also perform exceptionally well.
